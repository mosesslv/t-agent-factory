# Failure Cases

这里存放 t-agent 的失败样例。

失败样例是产品资产，不是事故垃圾桶。每个失败样例都应帮助我们改进：

- 数据集语义。
- 指标口径。
- SQL 生成。
- SQL Guard。
- 报告引用。
- Agent 计划。
- 知识检索。

建议文件名：

```text
YYYY-MM-DD-GQ-001-short-title.md
```

最小结构：

```markdown
# 失败样例：标题

## 问题

## 数据集版本

## 实际输出

## 期望输出

## 失败类型

## 根因判断

## 修复动作

## 是否进入回归
```
