# t-agent Iteration Direction Research

## Quick Start

1. Read `agent.md`.
2. Read `09-agents/iteration-direction-research-protocol.md`.
3. Search local project context first.
4. Query ProductFactory when product judgment is involved.
5. Check external primary sources and relevant open-source/community signals.
6. Run resident-agent roundtable.
7. Classify recommendations as `Adopt now`, `Trial`, `Watch`, or `Avoid for now`.
8. Update the smallest durable artifact.

## Acceptance

- Sources are explicit.
- Community signals are not treated as final evidence without stronger support.
- Recommendation is mapped to version/stage.
- Product direction changes create or update PDR/ADR/backlog/eval.
