# Iteration Inbox

Use this folder for rough, incomplete, or conflicting ideas.

Good examples:

- A new angle found while reading AI_DB_GPT `inspiration`.
- A fragment from `gather` that may become a report section.
- A disagreement between ChatReport and current t-agent V2 assumptions.
- A note that may later become PRD, ADR, eval, or report material.

Rules:

- One idea per file when possible.
- Keep source links explicit.
- Mark status as `raw`, `candidate`, or `needs-review`.
- Do not present inbox notes as accepted product direction.
