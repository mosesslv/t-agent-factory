# Drafts

Use this folder for theme-level working drafts.

Drafts are more structured than inbox notes, but not yet accepted product truth.

Examples:

- `ai-dbgpt-report-agent-lessons.md`
- `sales-operating-report-outline-v0.md`
- `chatreport-to-t-agent-object-mapping.md`
- `evidence-first-reporting-narrative.md`

Promotion rule:

```text
draft -> product decision / ADR / PRD / architecture contract / eval
```

Only promote a draft after it names evidence, assumptions, unknowns, and the target artifact.
