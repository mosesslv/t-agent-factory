# Review Queue

本目录存放尚未进入正式产品结论的候选内容。

适合放入这里的内容：

- 新想法。
- 未确认来源。
- 待评审竞品观察。
- 用户口头反馈。
- agent 评审发现。
- 失败样例。

不适合放入这里的内容：

- 已接受 PRD。
- 已接受 ADR。
- 已纳入 eval 的正式验收问题。
- 已发布 release note。

晋升路径：

```text
review-queue
  -> evidence-card
  -> product-decision / ADR
  -> PRD / contract / eval / backlog
```
