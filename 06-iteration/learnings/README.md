# t-agent Learning Logs

本目录用于记录会改进 t-agent 工作台的学习事件。

它不是长期事实库，也不是聊天归档。它只保存需要后续复盘、晋升或修复的事件。

## 1. 可记录内容

- 用户纠正；
- agent 误判；
- 命令 / 工具 / 外部来源失败；
- 缺失能力；
- 重复出现的 workflow；
- 更好的知识更新模式；
- eval 失败样例。

## 2. 文件建议

| 文件 | 用途 |
|---|---|
| `YYYY-MM-DD-learning-events.md` | 当日学习事件 |
| `YYYY-MM-DD-errors.md` | 当日错误和工具失败 |
| `YYYY-MM-DD-feature-requests.md` | 当日能力请求 |

## 3. 晋升路径

```text
learning event
  -> review queue
  -> protocol / local skill / PDR / ADR / eval / backlog
```

## 4. 禁止内容

- secret、token、私密凭据；
- 未脱敏个人隐私；
- 完整聊天记录；
- 大段外部版权内容；
- 未经确认的结论直接标为 accepted。
