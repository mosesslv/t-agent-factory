# AI Data Report Prototype

PC 端 AI 数据报告产品原型，所有项目文件都放在当前目录内。

## 本地运行

```powershell
npm start
```

打开：

```text
http://127.0.0.1:4173
```

## 文件

- `index.html`：高保真单页原型
- `server.js`：本地静态预览服务
- `package.json`：项目脚本

当前版本不需要虚拟环境或外部依赖。后续如果加入 Python 数据分析服务，建议在本目录下创建 `.venv`；如果升级成前端工程，依赖也放在本目录的 `node_modules`。
