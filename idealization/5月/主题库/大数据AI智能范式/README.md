## 本库核心判断

未来企业数据平台的主线不是“数据仓库 + BI 报表”继续线性升级，而是转向：

```text
Governed Data
+ Business Context
+ Semantic Layer
+ Agent Runtime
+ Tool / Action Governance
+ Run Trace / Eval
+ Business Applications
```

大数据团队的职责也会从“数据开发与报表交付”转向“企业智能分析能力提供方”：提供可问数据集、指标语义、知识上下文、受控分析 Skill、Agent 行为治理、运行评测和分析资产沉淀。
