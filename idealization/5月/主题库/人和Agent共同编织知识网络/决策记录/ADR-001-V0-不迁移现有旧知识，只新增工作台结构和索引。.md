# ADR：V0 不迁移现有旧知识，只新增工作台结构和索引。

## 背景

本决策来自 `人和 Agent 共同编织知识网络` 的工作台沉淀候选。

## 决策

1. V0 不迁移现有旧知识，只新增工作台结构和索引。
2. V0 不引入 LangGraph、Mastra、Vercel AI SDK 或 Web App。
3. Promote 必须人工确认，不允许自动将工作台内容写入正式知识库。
4. 旧知识被替代时优先标记 deprecated，不直接删除。

## 影响

- `_workbench/` 保留研讨过程。
- `主题库/` 只接收确认后的正式知识。
- Promote 必须显式确认，并保留来源 session。

## 回退方式

- 通过 Git revert 回退本次 promote 提交。
- 如知识判断过时，优先标记 deprecated，而不是删除历史。
