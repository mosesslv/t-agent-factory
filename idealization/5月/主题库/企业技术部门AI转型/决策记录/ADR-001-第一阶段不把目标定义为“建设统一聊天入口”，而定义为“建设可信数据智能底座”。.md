# ADR：第一阶段不把目标定义为“建设统一聊天入口”，而定义为“建设可信数据智能底座”。

## 背景

本决策来自 `企业技术部门在 Agent 时代的定位` 的工作台沉淀候选。

## 决策

1. 第一阶段不把目标定义为“建设统一聊天入口”，而定义为“建设可信数据智能底座”。
2. 第一阶段选择企业数据分析作为样板场景，验证从可问数据集到反馈评测的完整闭环。
3. 不把未经验证的组织架构调整、具体 FTE 或框架选型写入正式知识。

## 影响

- `_workbench/` 保留研讨过程。
- `主题库/` 只接收确认后的正式知识。
- Promote 必须显式确认，并保留来源 session。

## 回退方式

- 通过 Git revert 回退本次 promote 提交。
- 如知识判断过时，优先标记 deprecated，而不是删除历史。
