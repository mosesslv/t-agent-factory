# Agent Knowledge

这里存放常驻产品 agents 共享或专属的系统知识。

示例：

- t-agent 产品定义摘要。
- V2 SSOT：Dataset Learning、Knowledge Base、ChatBI Adapter、ChatExcel 单文件/单表分析报告。
- 销售经营候选 golden workflow 上下文。
- 常用评审标准。
- 数据产品和 Agent Harness 术语表。

规则：

- 共享知识必须能追溯到来源或决策。
- 不确定内容先进入 `06-iteration/review-queue/`。
- 影响正式产品方向的内容进入 `05-decisions/`。
