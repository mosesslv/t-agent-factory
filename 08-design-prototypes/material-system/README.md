# Material System

This folder is the working area for HTML and visual material prototypes.

Use `.agents/skills/t-agent-material-strategy` before creating or reviewing materials here.

Recommended flow:

```text
source notes -> 06-iteration/inbox
storyline -> 06-iteration/drafts
report or deck outline -> 06-iteration/reports
HTML prototype -> 08-design-prototypes/material-system
accepted decision -> 05-decisions or 01-product/prd
```

Reference boundaries:

- AI_DB_GPT report-engine HTML can inform report quality expectations.
- Manus UI/UX references can inform task-first interaction principles.
- Do not copy third-party branding, screenshots, or distinctive layouts into deliverable material.
